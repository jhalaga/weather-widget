# Weather Widget
Android Weather Widget


### Build debug APK
```
.\gradlew.bat assembleDebug
```
APK will be created at:
```
app\build\outputs\apk\debug\app-debug.apk
```

### Build release APK  
```
.\gradlew.bat assembleRelease
```

### Clean project
```
.\gradlew.bat clean
```

### Install on connected device
```
.\gradlew.bat installDebug
```
